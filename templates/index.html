Welcoome! Use the API Endpoints to interact with the prediction business model.
API Endpoint Action Description
localhost:1313/predict GET Return the predicted outcome (business_outcome), predicted probability (phat), and all model inputs
/localhost1313/predict/predict POST This API allows you to feed in your variables and obtain the predictions and probabilities if the prediction is >= 0.75
localhost:1313/predict_all/predict POST This allows you to get all of the predictions for all of your input data
